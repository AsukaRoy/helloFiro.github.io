Week 1: Basic Elements of Photography:

Camera/Lens

Photography:

Create a image with light → Lens

Form & see light image → camera

Camera: the secret room of art

The pinhole principle → photograph

Lens: bring a brunch of light together at the focal plane. If the light are not at the focal place, the picture is outfocus.

Lens Focal Length: The distance from the optical center of the lens to the focal place

  • Normal Focal length: normal size
    • Full Frame sensor 36mm * 24mm → Normal Focal length = 50mm
    • APS_C sensor 27mm * 23mm → 30mm
    • Four Thirds sensor 21mm*17mm → 25mm
  • Telephoto Focal length: (longer) big size
  • Wide Focal length: (shorter) small size

Visual Degree & Focal length

  • 15mm 180°
  • 35mm 120°
  • 70mm 68°
  • 200mm 24°

SENSOR/SHUTTER

Sensor: the light sensitive materials

  • Full Frame sensor 36mm * 24mm
  • APS_C sensor 27mm * 23mm
  • Four Thirds sensor 21mm*17mm

Shutter: control the time that light on sensitive materials

There are two shutter and move from up to time at different time. The difference of time is called shutter speed.

Shutter + Aperture = Exposure

Shutter speed relationship

$$1 = 8X \
1/2 = 4X \
1/4 = 2X\
1/8 = X\
1/15 = 1/2X\
1/30 = 1/4X\
1/60 = 1/8X\$$

Apereture size:

$f.stop = \frac{Focal_length}{Diameter_of_Lens}$ the aperture can change the diameter of lens.

$$f.2 = 8X \
f.2.8 = 4X \
f.4 = 2X \
f.5.6 = X \
f.8 = 1/2X \
f.11 = 1/4X \
f.16 = 1/8X $$

Workflow of Digital photography step by step

  1. Use your camera
  2. Memory card to reader
  3. Copy to computer
  4. Organize in software (Add keyword and star)
  5. Copy to backup drives
  6. upload to Cloud/Flickr/website
  7. Prep and print and save
  8. Repeat, make pictures

Full auto shooting mode

It releases you from setting and focus on shooting.
